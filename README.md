## About This Project

Its a simple quiz management using laravel 9 and livewire. this project has features such as:

- Register and login.
- User Management.
- Role and permission management.
- Material management.
- Question management.

## Installation

- clone this repo.
- composer install.
- php artisan key:generate.
- php artisan db:seed.
- npm install

## To Run It

- php artisan serve
- npm run dev or watch if needed
